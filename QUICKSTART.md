# 貿易DX Webアプリ - クイックスタートガイド

## 🚀 最速で起動する方法

### ステップ1: シードデータの投入（初回のみ）

```batch
cd backend
scripts\run_seed_simple.bat
```

### ステップ2: バックエンドの起動（ターミナル1）

```batch
cd backend
py -3 -m uvicorn app.main:app --reload
```

**確認:** ブラウザで http://localhost:8000/docs にアクセスしてAPI仕様が表示されることを確認

### ステップ3: フロントエンドの起動（ターミナル2）

**新しいターミナルを開いて：**

```batch
cd frontend
npm run dev
```

**確認:** ブラウザで http://localhost:3000 にアクセス

### ステップ4: ログイン

- **ユーザー名:** `admin`
- **パスワード:** `admin123`

## ❗ トラブルシューティング

### フロントエンドが真っ白

**原因:** バックエンドが起動していない

**解決策:**
1. バックエンドを先に起動する
2. http://localhost:8000/health にアクセスして確認
3. フロントエンドを再読み込み（Ctrl+R または F5）

### バッチファイルが文字化け

**解決策:** シンプル版を使用
```batch
cd backend
scripts\run_seed_simple.bat
```

### ポート番号が違う

- **フロントエンド:** 3000番ポート（5173ではない）
- **バックエンド:** 8000番ポート

### データベースにデータがない

**解決策:** シードデータを再投入
```batch
cd backend
py -3 -m scripts.seed_data
```
